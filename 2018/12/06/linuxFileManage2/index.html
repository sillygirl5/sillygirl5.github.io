Linux文件与目录管理

Posted on 2018-12-06 | | 浏览 次
Words count in article: | Reading time ≈

常用的处理目录的命令

  • cd：切换目录
  • pwd：显示当前目录
  • mkdir：建立一个新的目录
    • mkdir [mp] directory name
    • [m]：设置文件的权限
    • [p]：递归创建所有目录
  • rmdir：删除一个空的目录
    • [p]：连同上一级“空的”目录也一起删除

文件与目录管理

  • 文件与目录的查看：ls [-aAdfFhilnrRSt] 文件或目录名称
    • [a]:全部文件，包括隐藏的文件
    • [d]:仅列出文件本身，而不是列出目录内的文件数据
    • [l]:详细信息
  • 复制（cp）、删除（rm）与移动（mv）
    • cp [-adfilprsu] source destiation
      • [i]：若目标文件已经存在，在覆盖时先询问
      • [p]：连同文件的属性（权限，用户，时间）一起复制过去
      • [r]：递归复制
      • [s]:复制称为符号链接文件（symbolic link），亦称为快捷方式
        Note:复制时需要考虑：
        • 是否需要完整的保留源文件信息？
        • 源文件是否为符号链接文件？
        • 源文件是否为特殊文件，例如FIFO、socket等？
        • 源文件是否为目录？
